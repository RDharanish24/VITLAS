## Installation Instructions

Install both frontend and backend dependencies using npm:

### Frontend
cd app
npm install

### Backend
cd backend
npm install

---

## Node.js Version
Recommended: Node.js v16 or higher

## Package Manager
npm, yarn, or pnpm (any modern package manager works)

## Notes
- This project uses npm/yarn package management, not pip
- All dependencies are specified in respective package.json files
- For production builds, use: npm run build
- For development, use: npm run dev
